# Ejercicios de Programación en Python

## 0. FizzBuzz
Escribe un programa que recorra del 1 al 100:
- Si el número es múltiplo de 3 → imprime **Fizz**
- Si es múltiplo de 5 → imprime **Buzz**
- Si es múltiplo de ambos → **FizzBuzz**
- En otro caso imprime el número.

## 1. Clasificador de números (listas + condicionales)
Dada una lista de números enteros, crea tres listas nuevas:
- `positivos`
- `negativos`
- `ceros`

Llena cada una recorriendo la lista original y clasifica cada número según corresponda. Imprime las tres listas al final.


## 2. Contador de vocales (funciones)
Escribe una función llamada `contar_vocales(texto)` que:
- reciba un string
- cuente cuántas vocales contiene
- devuelva un diccionario como:
```python
{"a": 3, "e": 1, "i": 0, "o": 2, "u": 0}
```

## 3. Lista de tareas (listas + diccionarios)
Crea un programa que permita manejar una lista de tareas.  
Cada tarea debe ser un diccionario con:
- `"titulo"`
- `"completada"` (True/False)

Permite:
- agregar una tarea
- marcar una tarea como completada por índice
- mostrar todas las tareas con un formato claro

## 4. Máquina de bebidas (condicionales + loops)
Simula una máquina que ofrece:
1. Agua – $1  
2. Jugo – $2  
3. Café – $3  

El programa debe:
- mostrar el menú
- pedir al usuario que elija
- pedir el pago
- validar si el dinero es suficiente
- devolver cambio si aplica
- imprimir un mensaje de compra exitosa
